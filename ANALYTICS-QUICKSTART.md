# Analytics Quick Start

**30-Minute Implementation Checklist**

---

## Prerequisites
- [ ] Access to: https://mikejones.online/ghost
- [ ] Admin credentials ready

---

## Step 1: Verify Ghost Analytics (5 min)

1. Go to: https://mikejones.online/ghost
2. Click **Analytics** in left sidebar
3. Verify dashboard is visible
4. Note current metrics (or zeros if new)

**✓ Expected:** Analytics dashboard showing site metrics

---

## Step 2: Configure Resume Tracking (10 min)

**Simple Approach (Recommended):**

1. Verify resume page exists at `/resume/`
2. No code needed - Ghost tracks page views automatically
3. Check: Analytics → Pages → Find "Resume" page
4. Views = download interest

**Done!** No custom code required.

---

## Step 3: Test Analytics (10 min)

1. **Open incognito window**
2. **Visit pages:**
   - Homepage
   - About page
   - Resume page
   - 1-2 project pages
3. **Check dashboard:**
   - Go back to Analytics
   - Wait 1-2 minutes
   - Verify visitor count increased
   - Check page views incremented

---

## Step 4: Bookmark & Monitor (5 min)

1. **Bookmark:** https://mikejones.online/ghost (Analytics section)
2. **Weekly monitoring:**
   - Visitors (total traffic)
   - Top posts/pages
   - Resume page views
3. **Monthly:** Review trends, adjust content strategy

---
