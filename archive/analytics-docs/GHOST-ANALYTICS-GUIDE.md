## Troubleshooting

### Analytics Not Showing Data?

1. **Wait 5 minutes** - Analytics aren't real-time
2. **Check incognito mode** - You may be filtered
3. **Verify site is published** - Draft sites don't track
4. **Clear browser cache** - Refresh dashboard

### Can't Access Dashboard?

1. **Check login credentials** - Ensure you're logged in
2. **Verify URL** - Use correct Ghost admin URL
3. **Check permissions** - Ensure you have admin/owner role

### Zero Visitors Showing?

- **Normal for new sites** - Takes time to build traffic
- **Check after testing** - Use incognito to test
- **Verify tracking is on** - Should be enabled by default
