## The Easiest Path

1. **Open Ghost Admin:**
   ```
   https://mikejones.online/ghost
   ```

2. **Click "Analytics" in left sidebar**
   - You'll see the analytics dashboard
   - It's already collecting data (if you've had visitors)

3. **Verify resume page tracking:**
   - Go to: Analytics → Pages
   - Find "Resume" page
   - Page views = download interest

4. **Test with incognito window:**
   - Open private/incognito browser
   - Visit your site, resume page, project pages
   - Go back to Analytics dashboard
   - Wait 1-2 minutes
   - Verify views increased

**Done!** That's all you need for MVP.
