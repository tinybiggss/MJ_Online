## If You Encounter Issues

### Issue 1: Can't find dark mode settings
**Solution:** Look for "Color Scheme" or "Theme Settings" in Design → Kyoto settings

### Issue 2: Accent color not showing
**Solution:**
- Verify hex format: #4F46E5 (include the #)
- Save and hard reload browser (Cmd+Shift+R)

### Issue 3: Homepage layout unchanged
**Solution:** Check if Kyoto requires custom homepage page template

### Issue 4: Navigation not saving
**Solution:** Check for special characters, ensure URLs are valid

**Full troubleshooting:** See customization-guide.md (Part 10)
