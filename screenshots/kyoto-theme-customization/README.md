## Screenshot Guidelines

### Tools to Use:
- **Full-page:** Browser extensions like "Full Page Screen Capture"
- **Mobile:** Chrome DevTools → Device Mode → Capture screenshot
- **Sections:** Standard screenshot tool (Cmd+Shift+4 on Mac, Snipping Tool on Windows)

### Naming Convention:
Use the exact filenames listed above for consistency.

### Image Specifications:
- **Format:** PNG (for quality and transparency)
- **Desktop width:** 1920px or your actual screen width
- **Mobile width:** 375px (iPhone) or 414px (iPhone Plus)
- **Compression:** Optimize images but maintain readability

## How to Take Screenshots

### Full-Page Homepage (Desktop):

**Method 1: Chrome DevTools**
1. Open https://mikejones.online
2. Press F12 to open DevTools
3. Press Ctrl+Shift+P (Cmd+Shift+P on Mac)
4. Type "Capture full size screenshot"
5. Press Enter
6. Save as `homepage-desktop-full.png`

**Method 2: Browser Extension**
1. Install "Full Page Screen Capture" or similar
2. Visit https://mikejones.online
3. Click extension icon
4. Save as `homepage-desktop-full.png`

---

### Mobile Screenshots:

1. Open https://mikejones.online in Chrome
2. Press F12 to open DevTools
3. Click "Toggle device toolbar" icon (Ctrl+Shift+M)
4. Select "iPhone 12 Pro" or similar device
5. Press Ctrl+Shift+P and type "Capture screenshot"
6. Save as `homepage-mobile.png`

---

### Design Settings Panel:

1. Log into Ghost Admin: https://mikejones-online.ghost.io/ghost/
2. Go to Settings → Design
3. Use standard screenshot tool to capture full panel
4. Include as much of the settings as possible in one image
5. Save as `design-settings-panel.png`
