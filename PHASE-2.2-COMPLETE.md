## Design Decisions Summary

All decisions made autonomously as instructed:

### 1. Dark Mode: Onyx Preset ✅
**Choice:** Kyoto's "Onyx" dark mode preset
**Why:** Professional tech aesthetic, excellent readability, modern AI/ML appearance
**Alternatives Rejected:** Pure (too stark), Fossil (less tech-focused), Ice (too cold)

### 2. Accent Color: #4F46E5 (Indigo) ✅
**Choice:** Indigo (#4F46E5)
**Why:** Tech industry standard, excellent contrast (7.12:1 ratio on dark), WCAG AA compliant
**Alternatives Rejected:** #3B82F6 (too traditional), #8B5CF6 (too playful), #10B981 (less tech-focused)

### 3. Typography: Kyoto Defaults ✅
**Choice:** Keep Kyoto's default fonts
**Why:** Already optimized for dev portfolios, professional appearance, saves time
**Customization:** None needed unless specific branding requirement arises later

### 4. Homepage Layout: Portfolio Focus ✅
**Structure:**
- Hero section: "AI/ML Engineer & Researcher" + CTA
- Featured projects: Grid/cards, 3-4 projects
- Recent writing: List, 3 posts
**Why:** Showcases technical expertise immediately, professional standard

### 5. Logo/Branding: Text-Based ✅
**Choice:** Text "Michael Jones" (no custom graphic)
**Why:** Clean, professional, minimal aesthetic, can upgrade later
**Implementation:** Simple, fast, ages well

### 6. Navigation: 5 Primary + 2 Secondary ✅
**Primary:** Home, Projects, Writing, About, Resume
**Secondary (Footer):** Contact, RSS
**Why:** Comprehensive coverage, clear hierarchy, not crowded
